## 名称
mymd — Markdown Preview

## 简短描述 (132字符以内)
Beautiful, customizable Markdown preview with 8 themes, code highlighting, math, diagrams, and bilingual support.

## 详细描述

mymd turns your browser into a beautiful Markdown reader.

Open any .md file — locally or online — and see it rendered with professional typography, syntax highlighting, and your choice of theme.

✨ Features:
• Rich rendering: CommonMark + GFM, LaTeX math (KaTeX), Mermaid diagrams, 14+ syntax extensions
• 8 built-in themes: GitHub, Typora, Academic, Reader, Minimal, Handwriting, Cyberpunk, Retro
• Shiki code highlighting with VS Code themes
• Floating glass sidebar with outline navigation
• Same-directory file browsing for local files
• Source view with line numbers
• Auto-refresh on file changes
• Custom CSS and CSS variable tuning
• Theme import/export (JSON)
• Print/PDF optimized
• Bilingual UI (Chinese / English)
• Keyboard shortcuts

🎨 Highly Customizable:
Every visual detail is controlled by CSS variables. Choose a theme, fine-tune colors, inject custom CSS, or create and share your own themes.

🔒 Privacy:
mymd runs entirely locally. No data is sent to any server. No analytics, no tracking, no account required.

📖 Open Source:
https://github.com/chasey-myagi/mymd

## 权限说明 (审核时可能需要)
- storage: 保存用户设置和主题配置
- activeTab: 检测当前页面是否为 Markdown 文件
- webNavigation: 拦截 .md URL 并重定向到渲染页面
- tabs: 在设置中打开测试页面
- file:///: 预览本地 Markdown 文件
- http/https: 预览在线 Markdown 文件
